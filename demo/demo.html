CalTip Demos

What did you say world? You want another jQuery tooltip? You got it.

Demo 1: No arguments

If nothing is specified, the script uses the text from the data-tooltip attribute (if present), or if not present, then it will use the text from the actual element as a fallback.

$('a').caltip();
Demo 2: Simple string argument

You can provide a simple string of text.

$('a').caltip('This is showing an example for demo 2.');
Demo 3: Typical implementation with content and title

This is probably the most typical implementation of the tooltip. Just provide a title and content.

$('a').caltip({
  content :'This is probably the most common implementation', 
  title   :'Typical Example'
});
Demo 4: Custom

This example shows all of the various options implemented, including an optional callback function which executes after the tooltip has been activated. In this example, it turns the link CSS to red. It also demonstrates the offset property, which pushes the tooltip from its default location.

$('a').caltip({
  titlefont   : 'bold 20px/20px serif',
  font        : 'normal 12px/16px serif',
  padding     : '15px',
  titlecolor  : '#ff0000',
  bodycolor   : '#000000',
  border      : '2px solid #000',
  background  : '#9fa6de',
  boxshadow   : '0 10px 17px rgba(0, 0, 0, 0.5)',
  zindex      : 9999,
  width       : 400,
  height      : 'auto',
  title       : 'A custom tooltip',
  content     : 'This tool tip has been customized in every way possible',
  notch       : true, //coming soon!
  dataAttr    : false,
  offsetleft  : 245,
  offsettop   : 55,
  mouseevent  : 'mouseover',
  callback    : function(){ 
    $('#fourthexample').css('color', 'red'); 
  }
});
Demo 5: Change Content after initializing CalTip

In this example, we change the tooltip content after the initial call to the tooltip. Look at the tooltip, then click the button, and see that the tooltip content and title have changed. You will need to call this twice in separate statement (no chaining) or you can assign to a variable.

$('a').caltip({
  content : 'Some example initial content here.',
  title   : 'Here is the initial title'
});
$('button').on('click', function(){
  $('a').data('caltip').changeTitle('New Title!!!!');
  $('a').data('caltip').changeContent('And some new content!!!');
});
